📌 Вимоги:

firmware.elf має бути зібраний PlatformIO
Файл backtrace.txt має бути у тій самій папці
decode_bt_auto.bat має бути у тій самій папці
Папка  підкаталозі .pio\build\...


✅ Так, файл decode_bt_auto.bat запускається подвійним кліком у Windows — він відкриє вікно командного рядка, автоматично:

Знайде .elf файл у підкаталозі .pio\build\...

Зчитає backtrace-адреси з файлу backtrace.txt

Запустить xtensa-esp32-elf-addr2line для декодування

Покажe результат у вікні

І залишить вікно відкритим (pause) — зручно для перегляду

у каталозі з .bat є файл backtrace.txt
